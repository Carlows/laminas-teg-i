Organigrama General de la Empresa

Productos y/o Servicios

Hash Labs C.A., es una empresa del sector de T.I (Tecnologías de Información), se dedica al desarrollo de aplicaciones web y móviles. Ofrecen servicios como creación de blogs, páginas web, tiendas online y aplicaciones móviles.

Mercado objetivo

El objetivo principal de Hash Labs C.A. son los emprendedores y empresarios, ya que son las personas que buscan y necesitan el tipo de servicios que desarrolla la empresa. Estos empresarios buscan contratar personas jóvenes que mantengan una visión fresca y desarrollen iniciativas de valor.

Objetivos del proyecto

Objetivo General

Desarrollar un sistema de información mediante el diseño e implementación de una interfaz de usuario que sea ágil, puntual y que funcione en cualquier tipo de dispositivo para la automatización del proceso de registro y generación de facturas de las horas de trabajo que la empresa Hash Labs C.A. invierte en sus proyectos.

Objetivos Específicos

  • Realizar un análisis de los requerimientos actuales de la empresa mediante la creación de las historias de usuario a fin de llevar control del plan de trabajo en el que se realizará el sistema de información.
  • Diseñar el esquema de los modelos de base de datos y las interfaces de usuario para la obtención de la estructura base del sistema de información.
  • Desarrollar el sistema de información y su base de datos junto con pruebas unitarias y de funcionalidad con el fin de que se garantice su correcto funcionamiento.

Clase TimeEntry en Ruby on Rails


class TimeEntry < ActiveRecord::Base
  belongs_to :project
  belongs_to :user

  validates :description, presence: true
  validates :project_id, presence: true
  validates :time, presence: true

  scope :order_latest, -> { order(created_at: :desc) }
  scope :include_projects, -> { includes(:project) }

  def duration_sec
    (self.time.end - self.time.begin).to_i
  end

  def duration_hours
    (duration_sec.to_f / 60) / 60
  end
  
  # more methods
end
